* Pre-requisites
** Raspberry
- [X] Raspbian: the official OS for the Raspberry Pi.
- [X] Internet connectivity: Ethernet or Wi-Fi
- [X] SSH access enabled
** Your machine
- [X] sshpass installed
- [X] Python 3.7+ and poetry installed
* Let's get started
